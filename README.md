# Superlekker


## Installing

Clone the repo and use bun.
```bash
bun install
bun run db create
bun run db up
bun run db seed
```

## Developing

Running the development server:

```bash
bun run dev
```

## Building

Building for production:
```bash
bun run build
```

Running the production build:

```bash
bun run .output/server/index.mjs
```
